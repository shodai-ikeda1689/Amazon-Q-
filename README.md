# ボールキャッチゲーム

シンプルなPygameで作られたボールキャッチゲームです。

## 遊び方

1. 画面下部の青い長方形がプレイヤーです
2. 上から落ちてくる赤いボールをキャッチしましょう
3. ボールをキャッチするとスコアが1点増えます
4. ボールを逃すとスコアが1点減ります
5. ボールをキャッチするごとに、ボールの速度が少し上がります

## 操作方法

- 左矢印キー: プレイヤーを左に移動
- 右矢印キー: プレイヤーを右に移動
- ESCキー または ウィンドウの×ボタン: ゲーム終了

## 実行方法

```
python3 ball_catch_game.py
```

## 必要条件

- Python 3
- Pygame
